# 📄 ChatPDF - Chat with Your Documents

## 📌 About Project

ChatPDF is an AI-powered application that allows users to upload PDF documents and ask questions based on the content of the document.

The application uses Large Language Models (LLM) to understand the PDF content and provide relevant answers.

## ✨ Features

- 📂 Upload PDF documents
- 💬 Ask questions from uploaded PDF
- 🤖 AI-based responses
- 🔍 Document search and retrieval
- ⚡ Interactive user interface using Streamlit

## 🛠️ Technologies Used

- Python
- Streamlit
- LangChain
- ChromaDB
- Groq API
- Large Language Model (LLM)
- PyPDF Loader

## ⚙️ How It Works

1. User uploads a PDF file
2. PDF content is extracted and divided into smaller chunks
3. Document data is stored using vector database
4. User asks questions
5. AI retrieves relevant information and generates answers
